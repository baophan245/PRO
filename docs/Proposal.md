## 1. Introduction
This project aims to develop a simple Flappy Bird game using Java in order to practice object-oriented programming and game loop concepts.

## 2. Technologies Used
- Programming Language: Java
- IDE: NetBeans
- Libraries: Java Swing / AWT
- Version Control: GitHub

## 3. Main Features
Bird Control: The player presses the SPACE key to make the bird fly upward.

Gravity Effect: Gravity continuously pulls the bird downward.

Pipe Movement: Pipes move from right to left across the screen.

Random Pipe Gaps: The gaps between pipes are generated randomly to increase difficulty.

Continuous Scrolling: The game world scrolls endlessly to simulate forward movement.

Collision Detection: The system checks for collisions between the bird and pipes.

Ground Collision: The game ends if the bird touches the ground.

Score Tracking: The score increases when the bird successfully passes through pipes.

Game Over Screen: A message is displayed when the game ends.

Restart Option: The player can restart the game after losing.
